#Git questions
1. What is rebase vs merge?
2. what is hotfix?
3. what is cherrypick?
4. what are the types of branching strategies, and which one do you use?
5. what is .gitignore?


Answers :-
1. Merge creates it own history of branches while rebase joins the branch history into one repo.
2. Git hotfix is a specital type of branch used to quickly address and fix the critical bugs in live production environment.
3. Cherry-picking is the process of selecting a specific commit from one branch and applying to the another.
4. There are 3 types of branch strategies :- gitflow best for structured & multibranch, github flow best for simple & feature based, and trunk-based development and these can be used as per the size of the team.
5. .gitignore is a file in which we have to specifiy which directory or which file will not be added in the git repo.
