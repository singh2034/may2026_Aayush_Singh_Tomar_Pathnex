# What I did today?
1. Attended morning session from 6 AM to 8 AM on the topic of Monitoring the resources, - DataDog, Prometheus, Grafana
2. What is DataDog - it is a cloud based monitoring agent based tool which allows to monitor the resources and collects the logs of it for future use.
3. What is Prometheus - It is a open source systems monitoring and alterting toolkit which collects performance metrics from apps and infra such as CPU usage, RAM usage, ROM usage, and requests traffic.
4. What is Grafana - It is an open source data visualization and observability platform. It allows the user to visualize the real metrics in the form of UI so that it can be easily understandable.
