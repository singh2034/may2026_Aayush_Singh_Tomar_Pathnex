# Record types - Route 53
1. Record type is for pointing the ip to the domain.
2. Simple routing means simple. 
3. Weighted routing means condition where we can transfer the traffic of the domain e.g. 50-50 ratio or 60-70 ratio.
4. Geofencing routing - meant for ip routing for geographical area wise.
5. Web Proxy - meant for only web application - Traffic passes through main host and uses the web proxy to access the banned websites.
