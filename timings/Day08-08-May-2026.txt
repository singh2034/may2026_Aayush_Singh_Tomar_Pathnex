2. AWS Session Manager — Securely connect to EC2 instances without SSH keys or opening port 22.

	Example: Sending an alert when EC2 CPU usage exceeds 80%.

7. AWS Lambda — Runs serverless code without managing servers. 
	Example: Automatically resizing images uploaded to S3.

8. AWS Shield — Protects applications from DDoS attacks. 
	Example: Defending a public website from traffic flooding attacks.

9. AWS WAF (Web Application Firewall) — Filters and blocks malicious HTTP/HTTPS web traffic. 
	Example: Blocking SQL injection attacks on a web app.

10. Amazon GuardDuty — Threat detection service that monitors AWS accounts and workloads for suspicious activity. 
	Example: Detecting unauthorized crypto-mining activity on EC2.
