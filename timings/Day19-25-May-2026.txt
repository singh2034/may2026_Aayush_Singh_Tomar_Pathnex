# What I did today?

2. Learnt about what is CI/CD?
	- CI is continous integration that allows the users to upload or push their codes into a shared repository multiple times a day.
	- CD is Continuous Delivery that allows the users to deliver their code to environments like production, testing or debugging etc. with the help of humans.
	- CD is Continuous Deployment that allows the users to deploy their codes to production environments after all tests are passed but automatically.
3. What is GitLab CI/CD? - An application which allows us to use their platform to apply the CI/CD. It uses yaml language for the communication
4. What is Jenkins? - It is a free open source automation server that automates the software development lifecycle by allowing us to use the CI/CD. It uses groovy language for the communication.
5. What is Maven - It is a project management and build automation tool used primarly for java applications.
